Please note that these images won't work inside your local copy - this is intentional! They're meant to be viewable on GitHub.
# Setting Up:
#### Prerequisites:
1. [Git](https://git-scm.com/install/)
2. [Obsidian](https://obsidian.md/download)
#### Steps:
1. Open git bash.
2. Use the `cd` command to navigate to the location you want to clone this repository.
	1. `cd PATH/TO/YOUR/LOCATION/HERE`
		1. For example: if you want your location to be in a documents folder:
		2. `cd C:`
		3. `cd users/USERNAME_HERE/documents`
	2. If you made a mistake, you can `cd ..` to go up a layer.
		1. `cd ../../..` to go up multiple layers
3. `git clone "https://github.com/MrSkroob/mod-knowledge-base"`
4. Open the vault via Obsidian: ![](https://github.com/MrSkroob/mod-knowledge-base/blob/main/Pasted%20image%2020260313141841.png)
5. ![](https://github.com/MrSkroob/mod-knowledge-base/blob/main/Pasted%20image%2020260313141915.png)
